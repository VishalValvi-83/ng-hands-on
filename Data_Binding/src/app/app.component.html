Data Binding in Angular

1. String Interpolation

  • String interpolation binds data from the component to the template.
  • It uses double curly braces to display component properties.
  • Can display variables, method results, or simple expressions.
  • Use {{}} to bind data.
  • Example Code 
              app.component.html
          <h1>{{"{{title}}"}}</h1>  // Displays the title from the component
          <p>{{"{{2 + 2}}"}}</p>  // Output: 4

2. Property Binding

  • Binds component properties to HTML element properties.
  • Uses [property] syntax to bind data.
  • Can set attributes, properties, and CSS styles.
  • Use [property]="expression" to bind properties.
  • Example Code 
              
          <img [src]="logoURL">
          <button [disabled]="isButtonDisabled">Click Me</button>
          <input [value]="inputValue"/>

3. Event Binding

  • Binds component (TS) methods to HTML events.
  • Uses (event) syntax to handle events.
  • Can respond to user actions like clicks, key presses, etc.
  • Use (event)="methodName($event)" to bind events.
  • Example Code 
              app.component.html
            <button (click)="onButtonClick()">Click Me</button >
          <input (input)="onInputChange($event)" type="text" />

4. Two-Way Data Binding

  • Combines property and event binding for two-way data flow.
  • Uses [(ngModel)] syntax to bind data.
  • Requires importing FormsModule in the module.
  • Use [(ngModel)]="property" for two-way binding.
  • Example Code 
              
          <input [(ngModel)]="inputValue" />
          <p>Input Value: {{inputValue}}</p>
