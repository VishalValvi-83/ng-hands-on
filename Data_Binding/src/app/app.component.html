Data Binding in Angular

String Interpolation

  • String interpolation binds data from the component to the template.
  • It uses double curly braces to display component properties.
  • Can display variables, method results, or simple expressions.
  • Use {{}} to bind data.
  • Example Code 
    <!-- app.component.html -->
<h1>{{"{{title}}"}}</h1>  // Displays the title from the component
<p>{{"{{2 + 2}}"}}</p>  // Output: 4

Property Binding

  • Binds component properties to HTML element properties.
  • Uses [property] syntax to bind data.
  • Can set attributes, properties, and CSS styles.
  • Use [property]="expression" to bind properties.
  • Example Code 
    <!-- app.component.html -->
<img [src]="logoURL">
<button [disabled]="isButtonDisabled">Click Me</button>
<input [value]="inputValue"/>
